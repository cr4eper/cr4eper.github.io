python反序列化

此类序列化与反序列化与php的反序列化较为相似，相当于把程序运行时产生的变量，字典，对象实例等变换成字符串形式存储起来，以便后续调用，恢复保存前的状态

python中反序列化的库主要有两个，pickle和cPickle，这俩除了运行效率上有区别外，其他没啥区别

#序列化
import pickle
a_list = ['a','b','c']
print(pickle.dumps(a_list,protocol=0))
#运行结果：b'(lp0\nVa\np1\naVb\np2\naVc\np3\na.'


#反序列化
import pickle
a_list = ['a','b','c']
print(pickle.loads(b'(lp0\nVa\np1\naVb\np2\naVc\np3\na.'))
#运行结果：['a','b','c']

序列化方式

json格式

简介

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，易于阅读和编写，同时也易于机器解析和生成。JSON采用完全独立于编程语言的文本格式，这使得JSON成为理想的数据交换语言。JSON的基本结构是由键值对（key-value pairs）组成的对象（object）和数组（array）。

如果我们要在不同的编程语言之间传递对象，就必须把对象序列化为标准格式，比如XML，但更好的方法是序列化为JSON。因为JSON表示出来就是一个字符串，可以被所有语言读取，也可以方便地存储到磁盘或者通过网络传输。JSON不仅是标准格式，并且比XML更快，而且可以直接在Web页面中读取，非常方便。

JSON的两种结构

1、 “名称/值”对的集合（A collection of name/value pairs）

不同的编程语言中，它被理解为对象（object），纪录（record），结构（struct），字典（dictionary），哈希表（hash table），有键列表（keyed list），或者关联数组 （associative array）。

2、 值的有序列表（An ordered list of values）

在大部分语言中，它被实现为数组（array），矢量（vector），列表（list），序列（sequence）。

格式转换表

dump、dumps和load、loads

在此种反序列化方法中，前二者表示对其进行序列化，后二者表示反序列化，下面对其进行简单的代码实现。

dump与dumps

import json

# dumps可以格式化所有的基本数据类型为字符串
data1 = json.dumps([])         # 列表
print(data1, type(data1))
data2 = json.dumps(2)          # 数字
print(data2, type(data2))
data3 = json.dumps('3')        # 字符串
print(data3, type(data3))
dict = {"name": "Tom", "age": 18}   # 字典
data4 = json.dumps(dict)
print(data4, type(data4))

with open("test.json", "w", encoding='utf-8') as f:
    # indent 格式化保存字典，默认为None，小于0为零个空格。indent=4缩进4个空格
    f.write(json.dumps(dict, indent=4))
    json.dump(dict, f, indent=4)  # 传入文件描述符，和dumps一样的结果
    
#json文件当中的内容：
#{
#    "name": "Tom",
#    "age": 18
#}{
#    "name": "Tom",
#    "age": 18
#}
#输出的结果：
#[] <class 'str'>
#2 <class 'str'>
#"3" <class 'str'>
#{"name": "Tom", "age": 18} <class 'str'>

可以看出，在序列化之后的符号全是字符型，但都没有符号的变化，可以用于json的数据传输。

load和loads

import json


data5 = json.loads('{"name": "Tom", "age": 18}')
print(data5,type(data5))
#运行结果：{'name': 'Tom', 'age': 18} <class 'dict'>

可以看到，在经过序列化之后，他值得类型为字符，在反序列化之后，他的类型变为了字典。类型改变之后，便可以变回他原本的含义。

注意：json模块只能序列化基本的数据类型，如字符串、数字、列表和字典等

详细讲解见：Python序列化与反序列化详解（包括json和json模块详解）_Hardworking666的博客-CSDN博客

pickle格式

此格式与上述格式不同，在序列化之后会进行二进制序列化格式化，函数也是用dump、load。

import  os
import  pickle
import base64
class CTF():
    def __reduce__(self):
        return(os.popen,("calc",))
ctf = CTF()

bc = pickle.dumps(ctf)
print(bc)
#运行结果如下：b'\x80\x04\x95\x1b\x00\x00\x00\x00\x00\x00\x00\x8c\x02os\x94\x8c\x05popen\x94\x93\x94\x8c\x04calc\x94\x85\x94R\x94.'

在运行之后会将结果进行二进制转换，便于传输，上面的例子在反序列化之后会触发计算器，常用于检测代码是否会触发反序列化漏洞。

常用的命令执行的方法

__reduce__方法

当定义扩展类型时（也就是使用Python的C语言API实现的类型），如果你想pickle它们，你必须告诉Python如何pickle它们。 reduce 被定义之后，当对象被Pickle时就会被调用。它要么返回一个代表全局名称的字符串，Pyhton会查找它并pickle，要么返回一个元组。这个元组包含2到5个元素，其中包括：一个可调用的对象，用于重建对象时调用；一个参数元素，供那个可调用对象使用；被传递给 setstate 的状态（可选）；一个产生被pickle的列表元素的迭代器（可选）；一个产生被pickle的字典元素的迭代器。

import  os
import  pickle
import base64
class CTF():
    def __reduce__(self):
        return(os.popen,("calc",))
ctf = CTF()

bc = pickle.dumps(ctf)
print(bc)
#运行结果如下：b'\x80\x04\x95\x1b\x00\x00\x00\x00\x00\x00\x00\x8c\x02os\x94\x8c\x05popen\x94\x93\x94\x8c\x04calc\x94\x85\x94R\x94.'

在反序列话之后会执行__reduce__中的方法，便可以执行其中的方法。

map的用法：

map() 会根据提供的函数对指定序列做映射。第一个参数 function 以参数序列中的每一个元素调用 function 函数，返回包含每次 function 函数返回值的新列表。

如下例：

>>> def square(x) :            # 计算平方数
...     return x ** 2
...
>>> map(square, [1,2,3,4,5])   # 计算列表各个元素的平方
[1, 4, 9, 16, 25]
>>> map(lambda x: x ** 2, [1, 2, 3, 4, 5])  # 使用 lambda 匿名函数
[1, 4, 9, 16, 25]

# 提供了两个列表，对相同位置的列表数据进行相加
>>> map(lambda x, y: x + y, [1, 3, 5, 7, 9], [2, 4, 6, 8, 10])
[3, 7, 11, 15, 19]

map 函数在 Python 中是一种高阶函数，它接收一个函数和一个可迭代对象（如列表、元组、字典等）作为参数，然后对可迭代对象中的每个元素应用指定的函数，并返回一个迭代器。

简单来说便是可以接受一个参数，传到指定的函数当中来使用，这样边可以成功绕过改过滤。

C指令的妙用

当完全过滤R指令时，上面的方法便无法使用，这个时候便可以使用C指令，c指令，就是指用 C 语言编写的程序代码，可以使用 Python 的 ctypes 模块来调用 C 语言函数和操作 C 语言数据类型。通过 ctypes 模块，Python 程序可以像操作自己的内部数据类型一样操作 C 语言的数据类型和函数。
