go语言

基础介绍

注释

与python相同

package main
import "fmt"
func main(){
//单行注释
/*
多行注释
*/
fmt.Print("hello world?")
}

变量

var+变量名+变量类型，如：

var name String = "阿巴"

定义多个变量：

var{
    name String
    age int
}

注意，在定义后不使用会报错，在后边要使用才行。

初始化和赋值：

mane := "阿巴"

含义与上边相同。

变量交换

var a int = 100
var b int = 200
a , b = b , a
fmt.Print(a,b)

会输出a=200,b=100;

匿名变量

匿名变量就是一个下滑线_，本身即为一个特殊符号，赋给该变量的都会被抛弃，无法在接下来的代码中使用。

func GetData() (int, int) {
    return 100, 200
}
func main(){
    a, _ := GetData()
    _, b := GetData()
    fmt.Println(a, b)
}

像这样

作用域，与其他类型代码相同

常量

常量是一个简单的表示符，与其他语言当中的常量大致相同

const b string = "abc"

常量中只能是字符型，数字型，布尔型。

iota

特殊常量，可以认为是一个被编译器修改的常量，常量技术器
