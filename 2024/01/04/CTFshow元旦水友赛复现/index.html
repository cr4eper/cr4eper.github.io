CTFshow元旦水友赛复现

easy_include

打开之后直接给了一段代码：

<?php

function waf($path){
    $path = str_replace(".","",$path);
    return preg_match("/^[a-z]+/",$path);
}

if(waf($_POST[1])){
    include "file://".$_POST[1];
}

大概的意思就是，如果在传入的字符串中在消除.这个符号之后，第一个或者第一串是由字母组成的就将传入的参数拼接到file://这段字符串后边，然后用include包含该文件。
